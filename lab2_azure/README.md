# Лабораторная работа №2

## Сравнение сервисов Amazon Web Services и Microsoft Azure  
Создание единой кросс-провайдерной сервисной модели

---

## 1. Цель работы

Целью лабораторной работы является формирование навыков анализа и сопоставления публичных облачных сервисов различных провайдеров без привязки к конкретному вендору.

В ходе выполнения работы требовалось:
- использовать сервисную модель, разработанную в лабораторной работе №1;
- классифицировать потребление сервисов Microsoft Azure;
- сопоставить сервисы Azure и AWS на единой иерархии;
- сформировать кросс-провайдерную модель облачных сервисов.

---

## 2. Исходные данные

В качестве исходных данных использовались:
- результаты лабораторной работы №1;
- файл биллинга Microsoft Azure в формате CSV;
- шаблоны Meter Category, Meter Sub-Category, Meter Name и Consumed Service;
- документация Microsoft Azure.

Символ `%` в значениях Meter Name означает наличие произвольных символов до или после указанной строки.

---

## 3. Ход работы

### 3.1 Импорт данных

Файл биллинга Azure был импортирован в табличный редактор с разделителем «точка с запятой».  
Данные были приведены к формату, совместимому с сервисной моделью AWS.

### 3.2 Классификация сервисов

Для каждой строки биллинга:
- определялся тип сервиса Azure;
- выполнялось сопоставление с сервисами AWS;
- заполнялись поля IT Tower, Service Family, Service Type, Service Sub Type и Service Usage Type;
- соблюдалась логика классификации, принятая в лабораторной работе №1.

---

## 4. Результат

В результате была сформирована единая кросс-провайдерная сервисная модель, позволяющая:
- анализировать потребление облачных сервисов Azure;
- сопоставлять сервисы AWS и Azure;
- агрегировать данные от общего уровня до конкретных метрик потребления.

---

## 5. Вывод

В ходе лабораторной работы была успешно построена унифицированная модель классификации облачных сервисов, не зависящая от конкретного облачного провайдера.

Полученная модель обеспечивает целостное понимание структуры облачных сервисов и позволяет проводить аналитическое сравнение различных облачных платформ.
